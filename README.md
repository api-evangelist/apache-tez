# Apache Tez (apache-tez)
Apache Tez is an application framework that allows for a complex directed-acyclic-graph of tasks for processing data. It is built on top of Apache Hadoop YARN and is used as the execution engine for Apache Hive and Apache Pig.

## APIs

### Apache Tez
Tez provides a Java API for defining complex DAGs of data processing tasks, with vertex, edge, and data source/sink APIs for building execution plans that run on YARN, used as the execution engine for Hive and Pig.
